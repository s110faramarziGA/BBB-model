Scripts 
=======

The IFG scripts package in `/ifg/scripts` leverages excel data sheets, SMILES code csv files and the :ref:`chem-package-ref` 
to provide SMILES-based data processing and analysis scripts for users of the IFG package. The python scripts can be set up
for execution on your computer upon following the :ref:`installation-ref`. View the scripts below to see how to use them for 
SMILES-based functional group identification and analysis.
