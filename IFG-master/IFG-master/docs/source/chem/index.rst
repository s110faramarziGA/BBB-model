Chem Source Package
===================

The chem source package in `/ifg/chem` is a python package which contains the python source code for functional group identification. View 
any of the source code module references below to see their descriptions. If you would like to add more functional groups 
to the identification process, please add them to the `chem/data/functional_group_smiles_codes.csv` file in the repository in the format 
shown in the file.
